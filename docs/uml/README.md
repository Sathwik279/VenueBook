# EasyVenue UML Diagrams

This folder contains PlantUML source for the main views of the project:

- `usecase-diagram.puml`
- `architecture-diagram.puml`
- `sequence-booking-flow.puml`
- `activity-booking-process.puml`
- `class-diagram-core.puml`

These diagrams are based on the current code in:

- `Frontend/src`
- `Backend/src/main/java/com/easyvenue/backend`

They reflect the authenticated version of the application with JWT, role-based route protection, venue management, and booking workflows.
